# NUKE

## Installation

> Install nodejs before continuing

Install server dependencies

```
cd nuke
```

```
npm install
```

Install frontend dependencies

```
cd frontend
```

```
npm install
```

## Usage

To start the server and client at the same time (from the root of the project)

```
npm run dev
```
